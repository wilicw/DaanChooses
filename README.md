# DaanChoose
## Use Flask python3

### Requirements

- python 3
- nodejs
- npm (yarn)
- pip3
- mongoDB
- docker

### Get started

Clone this project

```
$ git clone https://github.com/wilicw/DaanChooses.git
```

Run backend server

```
$ cd server
$ source ./env/bin/activate
$ python main.py
```

Run ui

```
$ cd ui
$ yarn install
$ yarn serve
```
